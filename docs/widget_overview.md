# Widget Overview

Widgets are organized into categories called **widget groups**.

Access to a widget group automatically grants visibility of all the
widgets contained in that group.  
Widget groups are therefore used both to organize the dashboard
and to control user permissions.

The available groups cover different operational domains of the platform,
including infrastructure monitoring, network governance, service KPIs,
traffic analysis, and cost management.

Below is an overview of the main widget groups currently available.

---

# Infrastructure Monitoring

These groups provide visibility into the status and availability of
the IT infrastructure.  
They are typically used in operational dashboards to monitor system
health, availability, and operational anomalies.

- [IT Infrastructure](widget_it_infrastructure.md)  
  Infrastructure status and availability overview.

- [IT Analytics](widget_it_analytics.md)  
  Analytical views and historical insights about infrastructure behavior.

- [IT Anomalies](widget_it_anomalies.md)  
  Detection and reporting of abnormal infrastructure behaviors.

---

# Network Governance

These groups focus on monitoring and analyzing the Wide Area Network
and connectivity status.

- [Network](widget_network.md)  
  Core connectivity monitoring and network status information.

- [Network Analytics](widget_network_analytics.md)  
  Analytical views of network performance and connectivity metrics.

- [Net Flow](widget_net_flow.md)  
  Traffic analysis based on NetFlow data, providing insights into
  application-level network traffic.

---

# Service Management

These widgets provide insights into operational service performance
and support activities.

- [Support Service KPI](widget_support_service_kpi.md)  
  Key performance indicators related to service management activities,
  including event statistics and SLA compliance.

---

# Cost Governance

These widgets support financial monitoring of infrastructure resources.

- [Cloud Cost](widget_cloudcost.md)  
  Multi-cloud cost governance and expenditure monitoring.

- [Analytical Accounting](widget_analytical.md)  
  Analytical views of infrastructure-related costs.

---

# Dedicated / Customer Specific Widgets

Some widget groups are dedicated to specific environments or customers.

- [MCD Dedicated](widget_mcd_dedicated.md)  
  Custom widgets designed for the McDonalds environment.
