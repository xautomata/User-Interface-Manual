# Analytical Accounting

The **Analytical Accounting** widget group allows users to analyze cloud costs according to the hierarchical structures defined in **Cost Views**.

A single customer may define multiple Cost Views, each representing a different organizational perspective on cloud spending.  
For this reason, when an Analytical Accounting widget is configured for the first time, the user must select the **Cost View** that the widget will use as its reference structure.

These widgets are useful for exploring costs through a customized accounting model rather than only through the original billing structure provided by the cloud vendors.

## Cost Nodes Tree

Shows the entire cost node tree for the selected view.

## Cost Resources by Tag
